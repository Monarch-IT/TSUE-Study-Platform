# TSUE Study Platform — Galactic Voyage

Образовательная платформа, разработанная специально для Ташкентского Государственного Экономического Университета (TSUE). Платформа специализируется на обучении программированию на Python с глубокой геймификацией в космическом сеттинге, интеграцией ИИ (Monarch AI) и встроенной средой разработки (IDE).

## Описание проекта

TSUE Study Platform («Galactic Voyage») — это амбициозный проект, цель которого — сделать изучение программирования увлекательным и эффективным. Платформа сочетает в себе интерактивные курсы, практическую среду разработки и систему прогрессии, подобную RPG, чтобы мотивация студентов всегда оставалась на высоком уровне.

Центральной фигурой платформы является ИИ-наставник Monarch AI, который помогает студентам с теорией, объясняет ошибки в коде и мотивирует на дальнейшие достижения.

### Основные фичи:
*   **Monarch IDE:** Встроенная браузерная среда разработки с проверкой кода (AST), поддержкой графического вывода (turtle) и подсветкой синтаксиса. Студентам не нужно ничего устанавливать локально.
*   **Геймификация:** Космический сеттинг, система рангов (от Рекрута до Императора), квантовые монеты, достижения и квесты.
*   **Monarch AI:** Персональный ИИ-ментор, который адаптируется к уровню студента, проводит код-ревью и отвечает на вопросы 24/7.
*   **Кабинет преподавателя (Teacher Portal):** Продвинутая панель для управления студентами, проверки заданий, просмотра аналитики и логов активности в реальном времени.
*   **Мультиплеер-Квизы:** Соревновательный режим изучения теории в реальном времени.

## Технологический стек

### Frontend
*   **Фреймворк:** React 18, Vite
*   **Типизация:** TypeScript
*   **Стилизация:** Tailwind CSS, Shadcn UI
*   **Состояние:** React Query
*   **Роутинг:** React Router DOM
*   **3D и анимации:** Three.js (@react-three/fiber), Framer Motion
*   **Хуки форм:** React Hook Form, Zod

### Backend & AI
*   **Сервер:** Python 3.12, FastAPI
*   **База данных:** Supabase (PostgreSQL), Firebase
*   **ИИ интеграция:** Google Generative AI (Gemini Flash)
*   **Хостинг:** Render (Backend), Firebase Hosting (Frontend)

## Для студентов TSUE

Для того чтобы начать пользоваться платформой, вам необходимо:
1. Зарегистрироваться, используя корпоративную почту или выданные данные.
2. Пройти калибровку у Monarch AI.
3. Начать свое "галактическое путешествие" с первого модуля по Python!

## Запуск проекта (Development)

Для локального развертывания проекта необходимы Node.js и Python.

1.  Клонировать репозиторий.
2.  Установить зависимости фронтенда:
    ```bash
    npm install
    ```
3.  Настроить переменные окружения (см. файл `.env.example`).
4.  Запустить фронтенд:
    ```bash
    npm run dev
    ```
5.  Запустить бэкенд (в папке `backend`):
    ```bash
    pip install -r requirements.txt
    uvicorn main:app --reload
    ```
