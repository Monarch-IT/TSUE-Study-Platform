# TSUE Study Platform

Интерактивная образовательная система, разработанная для Ташкентского Государственного Экономического Университета (ТГЭУ). Платформа ориентирована на эффективное обучение основам программирования на языке Python с использованием технологий искусственного интеллекта и современных методов геймификации.

## Ключевые особенности

*   **Интегрированная среда разработки (Monarch IDE):** Позволяет студентам писать и исполнять код непосредственно в браузере без необходимости установки локального окружения. Поддерживает визуализацию работы алгоритмов.
*   **ИИ-ассистент (Monarch AI):** Интеллектуальный помощник на базе Gemini, который помогает студентам разбираться в теории, анализирует ошибки в коде и предоставляет персональные рекомендации по обучению.
*   **Система управления обучением:** Раздельные интерфейсы для студентов и преподавателей. Преподаватели могут назначать задания, отслеживать прогресс группы и анализировать активность каждого студента.
*   **Геймификация:** Процесс обучения представлен в виде космической экспедиции, где за выполнение заданий студенты получают достижения и повышают свой рейтинг.
*   **Соревновательный режим:** Модуль многопользовательских квизов для закрепления пройденного материала в интерактивной форме.

## Технический стек

### Фронтенд
*   React, TypeScript, Vite
*   Tailwind CSS, Shadcn UI
*   React Query для управления состоянием данных
*   Three.js и Framer Motion для визуальных эффектов и анимаций

### Бэкенд и инфраструктура
*   FastAPI (Python) для API-сервисов
*   Supabase / PostgreSQL для хранения данных
*   Firebase для авторизации и хостинга фронтенда
*   Интеграция с Google Generative AI

## Инструкция по развертыванию

### Требования
*   Node.js (версии 18 и выше)
*   Python 3.10+

### Локальный запуск
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Monarch-IT/TSUE-Study-Platform.git
   ```
2. Установка зависимостей фронтенда:
   ```bash
   npm install
   ```
3. Настройте переменные окружения в файле `.env` (используйте `.env.example` как шаблон).
4. Запуск в режиме разработки:
   ```bash
   npm run dev
   ```
5. Для запуска бэкенда перейдите в директорию `backend`, установите зависимости из `requirements.txt` и запустите сервер:
   ```bash
   pip install -r requirements.txt
   uvicorn main:app --reload
   ```
